Module 4 - horizontal clustering of variants¶


HorizontalClustering.horizontalClustering.clusterDetection(threshold)[source]¶

clusters the variants based on a given threshold; to do so, edges with a weight above the threshold are deleted from the given graph respresenting the optimal mappings

Parameters:threshold – the variant threshold the algorithm should use
Returns:list of subgraphs where each subgraph represents a cluster of variants
HorizontalClustering.horizontalClustering.horizontalRefinement(candidateLabels, graphList)[source]¶

Performs horizontal relabelling of event labels within a cluster; each event that belongs to the candidate labels will get a unique new label per cluster

Parameters:
  • candidateLabels – a list of labels that should be refined
  • graphList – a list of subgraphs where each subgraph represents a cluster of variants
Returns:

a list of refined subgraphs, where the attribute ‘newLabel’ is changed for each candidate label, such that the event labels are unique per cluster
